# TruMed - Frontend

Web Admin

## Requirements

```bash
* nvm
* nodejs ~ 16
* yarn ~ 1.22.19
```

## First Steps

```bash
1. Request access to AWS code commit (IAM)
2. Install the requirements on your local pc
```


## Installation

```bash
$ cp .env.local .env
$ nvm use 16
$ yarn
```

## Running the app

```bash
# development
$ nvm use 16
$ yarn start

```

## Test

```bash
# unit tests and coverage
$ yarn test
```

## Build

```bash
# build app for deployment
$ yarn build
```
